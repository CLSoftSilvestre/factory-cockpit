# FACTORY COCKPIT
Factory Cockpit is an intuitive dashboarding application designed specifically for Siemens Edge Devices. It enables real-time visualization and analysis of aggregated data streams from IIH (Industrial IoT Hub) essentials, empowering manufacturing and operations teams to make informed, data-driven decisions on the shop floor.

With Factory Cockpit, users can effortlessly monitor KPIs, machine health, performance trends, energy consumption, and other critical metrics, all delivered through highly customizable and role-based dashboards. 

The platform seamlessly collects live data from connected devices, offering clear visual insights without the need for complex configuration or data modeling.

Whether you’re a plant manager, maintenance engineer, or quality analyst, Factory Cockpit provides the transparency and flexibility you need to optimize factory operations—turning raw IIH essentials data into actionable information at the edge.


## Overview

![alt text](./screenshots/Dashboard_Demo_1.jpeg?raw=true)

The application allows the creation of several dashboards with several widgets each. The widgets can display the actual data of an attribute of aggregated data in a period of time:

The aggregation types availables are:
- None
- Count
- Average
- Min
- Max
- Sum
- First
- Last
- Standard Deviation
- Counter
- Power to Energy
- Energy to Power
- Flow to Amount
- Amount to Flow
- Time Weighted Average

### Types of widgets

![alt text](./screenshots/Dashboard_Demo_2.jpeg?raw=true)

The available widgets are:

- Chart (Line, bars, scatter)
- Card
- Indicator
- List

- Timeline
